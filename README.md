# 🔐 Node.js Authentication Backend API

A lightweight backend authentication system built with Node.js and Express.js using JWT-based security and middleware protection.

---

## 📱 Mobile Development Highlight

This project is fully created and tested on an **Android mobile device** using:

- Termux (Node.js runtime)
- Acode Editor (coding environment)
- Mobile API testing tools

👉 Entire development workflow completed without using any PC or laptop.

---

## 🚀 Key Features

- User authentication (Signup/Login)
- Secure JWT token implementation
- Middleware-based route protection
- Clean and simple backend structure
- Fast and efficient API design

---

## 🛠️ Technology Used

- Node.js
- Express.js
- JSON Web Token (JWT)
- bcryptjs
